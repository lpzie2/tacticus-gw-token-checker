# tacticus-gw-token-checker
how many gw tokens are left on both sides and who holds them. some basic stats as well.

15 Apr 2026: Added another performance stat (🔷). It's a weighted stat between Performance (⭐) and Attack Score (⚔️). Added automated season detection based on timestamps. Added some GW map:tile history to determine what maps the battles were fought on. This allows me to add map images and unit icons. Saw it first on murky's Tacticus Tracker (https://tacticus.xyz/) and I liked the look. Spent some time writing a hex grid mapper for Tacticus assets. Msg me if you want to use it. Added a demo button since I saw A_Tin_Can's RaidMan Replays (https://replays.tacticus-raidman.com/public/replays) use it and I liked the idea.

03 Apr 2026: Added battle details (⚔️,🛡️), sorting (🏆), a performance stat (⭐), default line tracking (🤖), more buttons to be mobile friendly. The performance stat was given to me by HarqAlAda. Still a lot more optimizations to do but it's in a decent enough state atm.
